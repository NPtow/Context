# JJFORRUSSIA · Пакет для интервью с кандидатами v2

## Что это

Обновлённый пакет после серии интервью `prodakt-1 ... prodakt-10`.

Главные изменения:
- меньше фиксированного опросника;
- сначала свободный рассказ, потом ситуативные добивающие;
- сначала определяется тип выхода на рынок, а не формальная роль;
- меньше споров про `product / project / operations`;
- меньше pitch, больше discovery.

## Как использовать

### До интервью

- открыть `protocol v2`;
- держать рядом `questions v2`;
- иметь под рукой `taxonomy`;
- не открывать preferences survey во время основного разговора.

### Во время интервью

- держать темп;
- не читать вопросник как анкету;
- сначала понять, как кандидат реально выходит на рынок;
- не спорить с кандидатом о названии его роли;
- выбирать follow-up вопросы под его способ поиска;
- собирать цифры только если у человека вообще есть числовой процесс;
- только потом делать short concept test.

### После интервью

- заполнить intake fields;
- прогнать readout через `analysis method v1`, если интервью даёт новый signal для волны;
- выбрать `вести / отложить / не вести`;
- если кандидат в фокусе, отдельно отправить preferences survey;
- если есть запись, сохранить transcript и note в `10_meetings`.

## Когда использовать v1

`v1` можно открывать только как исторический слой.

Для новых интервью рабочая версия теперь `v2`.
