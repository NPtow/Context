# Local Skills Mirror

Здесь лежит sync-managed зеркало локальных skills из `/Users/NIKITA/.codex/skills`.

Правила:
- содержимое отдельных skill-папок можно читать как обычные repo-файлы;
- эту папку не редактировать вручную, если изменения должны переживать следующий sync;
- любые устойчивые ручные additions лучше класть в `../cloud/`.
