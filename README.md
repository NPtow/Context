# Context

LLM-first memory system для долгого хранения контекста по:
- Никите как пользователю и фаундеру;
- активным и прошлым проектам;
- доменным знаниям;
- решениям, гипотезам и урокам.

Если нужно быстро понять, с чего начинать чтение, открой [context-map.md](/tmp/Context/context-map.md).

## Принцип
- `актуальное` хранится отдельно от `логов`;
- `память о Никите` хранится отдельно от `проектов`;
- `доменное знание` переживает смерть отдельных проектов;
- `архив` не удаляет опыт, а убирает его из активного контекста.

## Базовые команды
- `обновись`
- `покажи контекст проекта`
- `что изменилось`
- `покажи память обо мне`
- `поставить задачу`
- `все задачи по проекту X`

Полный протокол лежит в [00_system/commands.md](/tmp/Context/00_system/commands.md).
